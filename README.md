# MoodMate – Emotion Based Music Recommendation System

MoodMate is an AI-powered music recommendation system that detects the user's emotional state and suggests suitable songs accordingly.

## Features
- Emotion detection using CNN
- Music recommendation based on detected mood
- Real-time emotion analysis

## Technologies Used
- Python
- TensorFlow / Keras
- CNN
- Flask
- OpenCV

## How It Works
1. The system captures facial expression.
2. A CNN model predicts the emotion.
3. Based on the detected emotion, the system recommends songs.

## Emotions Detected
- Happy
- Sad
- Angry
- Neutral
- Surprise
